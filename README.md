# 🔢 浮動小数点数・シフト演算学習アプリ

高校生向けのコンピュータ数値表現学習Webアプリケーション。シフト演算とIEEE 754標準の浮動小数点数の仕組みを、視覚的でインタラクティブな方法で段階的に学習できます。

## 🌟 特徴

### シフト演算学習 🔄
- **ビット操作の基本から応用まで**: 左シフト・右シフトの概念を理解
- **アニメーション付き可視化**: ビットの移動をリアルタイムで確認
- **ステップバイステップ解説**: 各演算を4つのステップで詳しく説明
- **練習問題**: 5問の段階的難易度の問題で理解を深める
- **浮動小数点数との関連**: 実用的な応用例を学習

### 浮動小数点数学習 🌊
- **IEEE 754標準の完全理解**: 32bit/64bit両対応
- **詳細な変換プロセス**: ステップ0〜4の段階的な説明
- **基数変換の可視化**: 10進数と2進数の変換を表形式で理解
- **正規化プロセス**: ビットシフトを活用した正規化を学習
- **実時間検証**: 変換結果の精度を即座に確認

### UI/UX 🎨
- **Tailwind CSSによるモダンデザイン**: 洗練された見た目
- **ダークモード対応**: 目に優しい表示切り替え
- **完全レスポンシブ**: スマートフォン〜PCまで対応
- **スムーズアニメーション**: 学習体験を向上
- **プログレス保存機能**: LocalStorageで学習履歴を記録

## 📚 学習内容

### 1. シフト演算
- **基本概念**: `<<` (左シフト), `>>` (右シフト)
- **数学的効果**: 2の累乗倍・除算
- **ビット可視化**: 8ビットでの移動表示
- **実用例**: 浮動小数点数の正規化での応用

### 2. 浮動小数点数
- **IEEE 754標準**: 業界標準の数値表現
- **32bit構造**: 符号部(1) + 指数部(8) + 仮数部(23)
- **64bit構造**: 符号部(1) + 指数部(11) + 仮数部(52)
- **バイアス表現**: 127 (32bit), 1023 (64bit)
