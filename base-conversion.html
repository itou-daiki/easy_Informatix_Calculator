基数変換学習

n進数からm進数への変換

基数変換とは？

基数変換は、異なる進数の間で数値を変換する方法です。
コンピュータサイエンスでは、2進数、8進数、10進数、16進数がよく使用されます。

主な進数

2進数（Binary）

使用する数字: 0, 1
コンピュータの基本

例: 1010₂ = 10₁₀

8進数（Octal）

使用する数字: 0-7
2進数3桁を1桁で表現

例: 12₈ = 10₁₀

10進数（Decimal）

使用する数字: 0-9
日常で使用する進数

例: 10₁₀

16進数（Hexadecimal）

使用する記号: 0-9, A-F
2進数4桁を1桁で表現

例: A₁₆ = 10₁₀
位取り記数法とは？

各桁の位置によって値が決まる表記法です。例えば10進数の「123」は：
1×10² + 2×10¹ + 3×10⁰ = 100 + 20 + 3 = 123

変換の仕組み

n進数 → 10進数

位取り記数法を使って計算します。

例: 1011₂ → 10進数
1×2³ + 0×2² + 1×2¹ + 1×2⁰ = 8 + 0 + 2 + 1 = 11₁₀

10進数 → n進数

変換先の基数で割り算を繰り返します。

例: 11₁₀ → 2進数
11 ÷ 2 = 5 余り 1
5 ÷ 2 = 2 余り 1
2 ÷ 2 = 1 余り 0
1 ÷ 2 = 0 余り 1
答え: 1011₂ (余りを逆順に読む)
