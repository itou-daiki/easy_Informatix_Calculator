ようこそ

このアプリでは、コンピュータで使われる数値表現の基礎を学ぶことができます。
シフト演算とIEEE 754浮動小数点数の仕組みを、ステップバイステップで理解しましょう。

📊 シフト演算

ビットを左右に移動させる高速演算を学びます。

  • ✓ 左シフト（<<）と右シフト（>>）
  • ✓ 2進数での計算理解
  • ✓ ビット移動の視覚化
  • ✓ 練習問題で実践
シフト演算を学ぶ

🌊 浮動小数点数

IEEE 754標準の浮動小数点数表現を学びます。

  • ✓ 16bit/32bit/64bit対応
  • ✓ 符号・指数・仮数の理解
  • ✓ 変換プロセスの詳細説明
  • ✓ 正負の実数対応
浮動小数点数を学ぶ
