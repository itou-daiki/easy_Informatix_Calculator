ようこそ

このアプリでは、高校「情報Ⅰ」で学ぶコンピュータの数値表現と計算技能の基礎を学べます。
基数変換から始まり、データ量、論理演算、シフト演算、2の補数、浮動小数点数まで、ステップバイステップで理解しましょう。

📚 推奨学習順序

番号順に学習すると、段階的に理解を深められます。各トピックは前のトピックの知識を基礎としています。

1

🔢 基数変換

n進数からm進数への変換を学びます。

  • ✓ 2/8/10/16進数対応
  • ✓ 位取り記数法の理解
  • ✓ 計算過程の可視化
  • ✓ 練習問題で実践
基数変換を学ぶ
2

💾 データ量

データ量の単位と計算方法を学びます。

  • ✓ bit/Byte/KB/MB/GB/TB
  • ✓ 画像データ量計算
  • ✓ 音声データ量計算
  • ✓ 動画データ量計算
データ量を学ぶ
3

⚡ 論理演算

AND、OR、NOT、XORなどの論理演算を学びます。

  • ✓ 6種類の論理演算
  • ✓ 真理値表の理解
  • ✓ ビット演算の可視化
  • ✓ 練習問題で実践
論理演算を学ぶ
4

📊 シフト演算

ビットを左右に移動させる高速演算を学びます。

  • ✓ 左シフトと右シフト
  • ✓ 2進数での計算理解
  • ✓ ビット移動の視覚化
  • ✓ 練習問題で実践
シフト演算を学ぶ
5

➖ 2の補数

負数の表現方法と加減算を学びます。

  • ✓ 負数の表現方法
  • ✓ 2の補数の計算
  • ✓ 加減算の仕組み
  • ✓ 練習問題で実践
2の補数を学ぶ
6

🌊 浮動小数点数

IEEE 754標準の浮動小数点数表現を学びます。

  • ✓ 16bit/32bit/64bit対応
  • ✓ 符号・指数・仮数の理解
  • ✓ 変換プロセスの詳細説明
  • ✓ 正負の実数対応
浮動小数点数を学ぶ

学習の流れ

1

基礎概念の理解

各ページの「説明」タブで基本的な概念を学習

2

実際に体験

計算ツールで数値を入力し、変換プロセスを確認

3

練習問題で定着

練習問題を解いて理解を深める

  • • コンピュータの仕組みに興味がある方
